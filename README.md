📩 SMS Spam Detection using Machine Learning
<br>
📌 Project Overview

This project implements a Spam Detection System for SMS messages using Machine Learning.
The system classifies messages into two categories:

Ham → Normal / Legitimate message

Spam → Unwanted / Promotional message

The model is trained using Naive Bayes (MultinomialNB) and text vectorization to convert SMS messages into numerical features.

🛠 Technologies Used

Python

Pandas – Data handling

NumPy – Numerical operations

Matplotlib – Visualization

Scikit-learn – Machine learning algorithms
